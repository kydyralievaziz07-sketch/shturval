# ⚙️ 04. Backend-разработчик

**Простыми словами:** делает «мотор» сайта — невидимую часть, которая считает,
хранит и выдаёт данные. То, что под капотом.

## Что делает для «Штурвала»
- Создаёт логику: посчитать выручку, прибыль, остатки
- Делает «мостики» (API), через которые сайт получает данные
- Обрабатывает заказы, возвраты, клиентов
- Отвечает за вход в систему и за то, чтобы данные не путались

## Когда зовём
Когда оживляем данные — переходим от «картинки» к реально работающему сайту.

## Мои реальные навыки за этим специалистом
- `backend-patterns` — архитектура серверной части
- `api-design` — правильные «мостики» для обмена данными
- `fastapi-patterns` / `nestjs-patterns` — каркасы для сервера
- `error-handling` — чтобы ошибки не ломали всё
- `nodejs` / `python-patterns` — языки серверной части
