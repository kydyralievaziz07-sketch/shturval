# 🚀 09. DevOps / специалист по запуску (деплой)

**Простыми словами:** отвечает за то, чтобы сайт работал не только на твоём Mac, а был
доступен по ссылке, делал резервные копии и обновлялся без сбоев.

## Что делает для «Штурвала»
- Публикует сайт в интернете (чтобы открывать с телефона)
- Настраивает резервные копии (GitHub уже подключён ✅)
- Делает безопасные обновления (можно откатиться назад)
- Следит, чтобы сайт не «падал»

## Когда зовём
Когда захотим открыть сайт по ссылке и при каждом обновлении версии.

## Мои реальные навыки за этим специалистом
- `deployment-patterns` — правильная публикация и откаты
- `github-ops` — работа с GitHub (репозиторий уже создан)
- `docker-patterns` — упаковка сайта для запуска где угодно
- `canary-watch` — проверка сайта после обновлений
- `git-workflow` — порядок в версиях и истории
