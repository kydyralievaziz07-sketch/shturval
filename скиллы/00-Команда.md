# 👷 Команда разработки «Штурвал»

Это 10 специалистов («разработчиков»), которые нужны для создания нашего сайта —
бизнес-пульта с интеграцией 1С, рынка и ИИ.

Каждый файл описывает: чем занимается специалист, что он делает для нашего проекта
и какие мои реальные навыки (skills) за ним стоят. Всех их «играю» я — это как
одна команда внутри одного исполнителя.

## Состав команды

| № | Специалист | За что отвечает |
|---|-----------|-----------------|
| 01 | 🧭 Архитектор проекта | общий план, структура, порядок шагов |
| 02 | 🎨 UI/UX дизайнер | как сайт выглядит и ощущается |
| 03 | 🖥️ Frontend-разработчик | то, что видно в браузере (экраны, кнопки) |
| 04 | ⚙️ Backend-разработчик | «мотор» сайта: логика и сервер |
| 05 | 🗄️ Архитектор баз данных | где и как хранятся товары, продажи, клиенты |
| 06 | 🔌 Специалист по интеграции 1С | связь сайта с твоей 1С |
| 07 | 🤖 AI-инженер | ИИ-помощник, подсказки, прогнозы |
| 08 | 📈 Специалист по аналитике/дашбордам | графики, отчёты, воронки |
| 09 | 🚀 DevOps / деплой | публикация сайта, резервные копии, обновления |
| 10 | 🛡️ QA-тестировщик и безопасность | чтобы ничего не ломалось и данные были защищены |

## Как мы их используем
Не все сразу. По шагам: сначала Архитектор + Дизайнер + Frontend (внешний вид),
потом Backend + Базы данных (оживляем данные), затем 1С + AI (интеграции),
в конце DevOps + QA (публикация и проверка).
