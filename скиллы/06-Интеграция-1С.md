# 🔌 06. Специалист по интеграции 1С

**Простыми словами:** соединяет наш сайт с твоей 1С, чтобы данные (товары, продажи,
деньги) автоматически перетекали в «Штурвал». Самый важный для тебя специалист.

## Что делает для «Штурвала»
- Настраивает связь сайта с 1С (она стоит у тебя на компьютере)
- Делает так, чтобы остатки и продажи обновлялись сами
- Решает вопрос постоянного доступа к компьютеру с 1С
- Следит, чтобы данные совпадали и не дублировались

## Когда зовём
На отдельном, более позднем этапе — это технически самая сложная часть.

## Мои реальные навыки за этим специалистом
- `api-connector-builder` — построение коннектора к внешней системе (1С)
- `api-design` — правильный обмен данными
- `backend-patterns` — приём и обработка данных из 1С
- `error-handling` — устойчивость, если 1С временно недоступна

ℹ️ 1С обычно отдаёт данные через OData/HTTP-сервисы или выгрузку — выберем способ,
когда дойдём до этого шага.
