# 🖥️ 03. Frontend-разработчик

**Простыми словами:** делает всё, что ты видишь и нажимаешь в браузере — экраны,
таблицы, кнопки, графики. «Витрина» сайта.

## Что делает для «Штурвала»
- Превращает дизайн в рабочие экраны
- Делает живые таблицы товаров и продаж, фильтры, поиск
- Рисует графики и показатели
- Следит, чтобы всё быстро открывалось и не тормозило

## Когда зовём
Постоянно, когда речь о видимой части сайта.

## Мои реальные навыки за этим специалистом
- `react-patterns` — построение интерфейса на React (современный стандарт)
- `frontend-patterns` — общие приёмы для фронтенда
- `react-performance` — чтобы сайт был быстрым
- `vite-patterns` / `nextjs-turbopack` — сборка и запуск проекта
- `react-testing` — проверка, что интерфейс работает правильно
