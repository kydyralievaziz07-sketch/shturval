# 🗄️ 05. Архитектор баз данных

**Простыми словами:** решает, где и как хранить все данные магазина (товары, продажи,
клиенты), чтобы они не терялись и быстро находились.

## Что делает для «Штурвала»
- Проектирует «склад данных»: какие таблицы, как связаны
- Делает так, чтобы поиск по большому каталогу был мгновенным
- Заботится о резервных копиях данных
- Готовит данные для отчётов и ИИ

## Когда зовём
Перед тем как хранить реальные товары и продажи — на этапе «оживления» сайта.

## Мои реальные навыки за этим специалистом
- `postgres-patterns` — надёжная база данных (PostgreSQL)
- `database-migrations` — безопасное изменение структуры без потери данных
- `prisma-patterns` — удобная работа с базой из кода
- `redis-patterns` — ускорение через кэш
- `clickhouse-io` — база для тяжёлой аналитики (если данных станет много)
