# 🧭 Shturval (Штурвал) — пульт управления магазином Бизмарт

Личный сайт-пульт для владельца: товары, продажи, рынок, финансы из 1С и ИИ-аналитика.

## Как открыть
Дважды кликнуть файл **index.html** — откроется в браузере. Ничего устанавливать не нужно.

## Что внутри (прототип v0.2)
- 📊 **Дашборд** — продажи сегодня, выручка, остатки, средний чек
- 📦 **Товары** — каталог и остатки
- 🛒 **Продажи** — продажи за день и последние сделки
- 🏪 **Рынок** — предложения в собственных каналах (Instagram/сайт/WhatsApp)
- 💰 **Финансы (1С)** — деньги, закуп, остатки
- 🤖 **ИИ-аналитика** — подсказки и прогнозы
- ⚙️ **Настройки** — доступ пока только у владельца

⚠️ Все данные сейчас **демонстрационные** (выдуманные).

## План развития (1–2 месяца, по шагам)
1. Завести реальные товары
2. Реальные продажи
3. Подключить 1С (стоит на компьютере владельца)
4. Выгрузка товара в каналы («рынок»)
5. Оживить ИИ на реальных данных
6. Личные кабинеты для других клиентов (в самом конце)
